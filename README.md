# Task Manager App

I developed this task management application using React Native for the PRITECH Technical Assessment.

## Features

Here are the features:

- You can view tasks

- Add new tasks easily

- Delete tasks you no longer need

- Mark tasks as complete or undo them

- There's a task details screen for more info

- Search for specific tasks quickly

- Filter tasks by status: All, Completed or Pending

- Switch between dark mode and light mode

- Tasks are saved even when you close the app thanks to AsyncStorage Persistence

- The app integrates with an API

## Technologies Used

I used these technologies:

- React Native for the app

- TypeScript for type checking

- React Navigation for screen navigation

- Context API for state management

- AsyncStorage for data persistence

## Installation

To get started:

- Run `npm install`, in your terminal

- Then run `npx expo start` to launch the app
